波（Wave）

    定義：

    波是能量在空間中傳播的方式，並不伴隨物質的整體移動。常見例子有水波、聲波、光波等。

    特性

  • 波長（λ）：兩個相鄰波峰（或波谷）之間的距離。
  • 頻率（f）：每秒振動的次數。
  • 振幅（A）：波動的最大位移。
  • 傳播速度（v）：波的移動速度，v = f × λ。

縱波（Longitudinal Wave）

    定義：

    粒子的振動方向與波的傳播方向相同的波。

    特性

  • 出現「壓縮」與「疏鬆」區。
  • 通常在固體、液體、氣體中都能傳播。

聲波（Sound Wave）

    定義：

    在介質（如空氣、水）中以縱波形式傳播的機械波，能引起人耳聽覺。

    特性

  • 是縱波。
  • 無法在真空中傳播。
  • 頻率範圍決定音高，人耳可聽範圍約 20Hz – 20kHz。

電磁波（Electromagnetic Wave）

    定義：

    由變化的電場與磁場互相垂直且與傳播方向垂直所構成的橫波。

    光波、無線電波、X 光、紫外線等。

    特性

  • 無需介質，能在真空中傳播。
  • 波速在真空中為光速（約 3×10⁸ m/s）。
  • 橫波性質：電場與磁場彼此垂直，且都垂直於波的傳播方向。

地震波（Seismic Waves）

    定義：

    地震波（Seismic Wave）是地震發生時，由地震能量所產生的波動，它們會在地殼中傳播，造成地面的搖晃。

    1.實體波（Body Wave）

  • 能在地球內部傳播的地震波。
  • 由地震彈性回跳直接產生，是每次地震都會出現的波形。

    • P波（Primary Wave / 縱波 / 壓縮波）：

  • 速度最快，約每秒6.5公里，是地震儀最早接收到的波。
  • 質點運動方向與波傳播方向平行，產生一疏一密的振動（類似音波）。
  • 可以在固體和液體中傳播。
  • 造成的震動主要是上下顛簸，雖然強度較小，但能提供短暫的預警時間。

    • S波（Secondary Wave / 橫波 / 剪力波）：

  • 速度次於P波，約每秒3.5公里，約為P波的0.58倍。
  • 質點運動方向與波傳播方向垂直，產生一上一下或一左一右的振動（類似繩波）。
  • 無法在液體或氣體中傳播（因此無法穿過地球外核）。
  • 振幅較大，約為P波的1.4倍。
  • 通常是造成地震破壞的主因，因為其水平向振動對建築物的破壞性較強。

    • 2.表面波（Surface Wave）：

  • 由實體波產生，主要沿著地表傳播，很少到達地表以下的地方。
  • 傳播速度最慢，但能量最大，對震源附近地區的破壞最嚴重。
  • 洛夫波（Love Wave / LQ波）：質點在水平面上做水平剪切運動。
  • 雷利波（Rayleigh Wave / LR波）：質點在垂直於波傳播方向的垂直面上沿橢圓形軌跡運動
    會造成地表的上下和前後的搖晃。

    • 3.尾波（Coda Wave）：

  • 是最後到達觀測儀器的波，由其他波與地形交互作用（散射）產生。
  • 研究尾波有助於了解地質結構。
